# Password Strength Analyzer

A full-stack web application that evaluates password strength using **React** and **FastAPI**. It analyzes password security, estimates crack time, provides improvement suggestions, and generates strong passwords through a modern and responsive interface.

---

## Features

- Password strength analysis
- Secure password generator
- Crack time estimation
- Personalized security suggestions
- Missing security criteria detection
- Circular strength gauge
- Show / Hide password
- Responsive user interface

## Installation

### Clone the repository

```bash
git clone https://github.com/Yosri-khedher/password-strength-analyzer.git
```

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
